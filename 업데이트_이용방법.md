# 업데이트 이용방법

이 문서는 Google Sheets + Apps Script 성적표 자동생성 시스템을 적용하고 운영하는 방법입니다.

## 1. Apps Script 코드 교체

Google Apps Script 편집기에서 `Code.gs` 전체를 아래 파일 내용으로 교체합니다.

```text
월드프랩/성적표출력_AppsScript/Code.gs
```

## 2. HTML 템플릿 교체

Apps Script 편집기에서 `ReportTemplate.html` 전체를 아래 파일 내용으로 교체합니다.

```text
월드프랩/성적표출력_AppsScript/ReportTemplate.html
```

## 3. 저장 후 Google Sheets 새로고침

Apps Script에서 저장한 뒤 Google Sheets를 새로고침합니다.

새 메뉴가 보여야 합니다.

```text
성적표 생성
```

## 4. 필요한 컬럼 추가/점검

Google Sheets 메뉴에서 실행합니다.

```text
성적표 생성 > 필요 컬럼 추가/점검
```

이 기능은 다음 작업을 자동으로 처리합니다.

- 필요한 성적입력 컬럼 추가
- 예전 컬럼명 변경
- 더 이상 쓰지 않는 `Teacher's Comment` 컬럼 삭제
- 점수 입력 컬럼 순서 정리
- Vocabulary/Homework/Participation 드롭다운 적용
- 반 이름 드롭다운 적용
- Grammar 평균그룹 드롭다운 적용
- 출력설정 탭 기본값 추가

## 5. 성적입력 주요 컬럼

성적 입력 과목 순서는 아래와 같습니다.

```text
Writing(Alpha) 점수
Grammar 점수
Reading 점수
Listening,Speaking 점수
```

점수가 비어 있는 과목은 PDF에서 자동 제외됩니다.

## 6. 태도 평가 입력

아래 세 항목은 드롭다운으로 선택합니다.

```text
Vocabulary
Homework
Participation
```

선택지는 다음과 같습니다.

```text
Need Improvement
Good
Perfect
```

## 7. Grammar 평균그룹 입력

`Grammar 평균그룹`은 드롭다운에서 선택합니다.

```text
A
B
C
```

Grammar 평균은 일반 반 이름이 아니라 이 Grammar 평균그룹 기준으로 계산됩니다.

## 8. 출력설정 수정

`출력설정` 탭에서 시험/시즌마다 바꿀 수 있습니다.

```text
성적표 제목
코멘트 섹션 제목
출력월
저장폴더명
파일명규칙
```

예시:

```text
English Achievement Report – Final Semester 2026
English Achievement Report – Midterm Semester 2027
```

## 9. PDF 생성

현재 선택한 학생만 생성:

```text
성적표 생성 > 현재 행 학생 PDF 생성
```

출력여부가 TRUE인 전체 학생 생성:

```text
성적표 생성 > 전체 학생 PDF 생성
```

반 이름으로 생성:

```text
성적표 생성 > 반 이름 입력해서 생성
```

## 10. 시상 순위표 생성

성적 입력 후 아래 메뉴를 실행합니다.

```text
성적표 생성 > 시상 순위표 생성
```

그러면 `시상순위` 탭이 생성/갱신됩니다.

`시상순위` 탭에는 반별로 학생들이 총점 높은 순서대로 표시됩니다.

동점자는 같은 순위로 처리됩니다.

## 11. 생성로그 확인

PDF 생성 결과는 `생성로그` 탭에서 확인할 수 있습니다.

성공한 경우 Google Drive PDF 링크가 기록됩니다.

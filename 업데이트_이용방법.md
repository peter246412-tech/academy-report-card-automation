# 업데이트 이용방법

이 문서는 Google Sheets + Apps Script 성적표 자동생성 시스템을 적용하고 운영하는 방법입니다.

## 1. Apps Script 코드 교체

Google Apps Script 편집기에서 `Code.gs` 전체를 아래 파일 내용으로 교체합니다.

```text
월드프랩/성적표출력_AppsScript/Code.gs
```

## 2. HTML 템플릿 교체

Apps Script 편집기에서 `ReportTemplate.html` 전체를 아래 파일 내용으로 교체합니다.

```text
월드프랩/성적표출력_AppsScript/ReportTemplate.html
```

## 3. 저장 후 Google Sheets 새로고침

Apps Script에서 저장한 뒤 Google Sheets를 새로고침합니다.

새 메뉴가 보여야 합니다.

```text
성적표 생성
```

## 4. 필요한 기능 추가/점검

Google Sheets 메뉴에서 실행합니다.

```text
성적표 생성 > 필요 기능 추가/점검
```

이 기능은 다음 작업을 자동으로 처리합니다.

- 필요한 성적입력 컬럼 추가
- 예전 컬럼명 변경
- 학생명단과 성적입력에 숨김 `학생ID` 추가 및 누락 ID 부여
- 점수 입력 컬럼 순서 정리
- Vocabulary/Homework/Participation 드롭다운 적용
- 반 이름 드롭다운 적용
- Grammar 평균그룹 드롭다운 적용
- 학생명단 자동 동기화 트리거 설치/점검
- 학생명단과 성적입력을 학원 반 순서로 정렬
- 출력설정 탭 기본값 추가

처음 실행할 때 권한 승인 창이 나오면 승인합니다. 자동 동기화 트리거 설치에는 이 승인이 필요합니다.

## 5. 학생명단 자동 동기화

평소에는 별도 버튼을 누를 필요가 없습니다.

`학생명단`에서 아래 내용을 수정하면 `성적입력`에 자동 반영됩니다.

```text
요일
시간
반 이름
학생 이름
학생 추가/삭제
```

자동 동기화 후 두 시트는 아래 반 순서로 정리됩니다.

```text
Beginner 1, 2, 3, 4
Pre-Inter 1, 2, 3, 4
Advanced 1, 2, 3, 4
PowerStation 1, 2
```

`학생명단 → 성적입력 지금 동기화`는 자동 반영이 지연되었을 때 사용하는 수동 복구 메뉴입니다. 매번 누르는 버튼이 아닙니다.

동기화는 기존 반 드롭다운 규칙을 다시 만들지 않으므로 설정해 둔 반별 색상이 유지됩니다.

학생명단에서 빠진 학생의 성적행에 점수, 평가, 코멘트, 노트 또는 수식이 있으면 자동 삭제하지 않고 동기화를 중단합니다. 안내된 행을 확인한 뒤 다시 실행합니다.

## 6. 성적입력 주요 컬럼

성적 입력 과목 순서는 아래와 같습니다.

```text
Writing(Alpha) 점수
Grammar 점수
Reading 점수
Listening,Speaking 점수
```

점수가 비어 있는 과목은 PDF에서 자동 제외됩니다.

## 7. 태도 평가 입력

아래 세 항목은 드롭다운으로 선택합니다.

```text
Vocabulary
Homework
Participation
```

선택지는 다음과 같습니다.

```text
Need Improvement
Good
Perfect
```

## 8. Grammar 평균그룹 입력

`Grammar 평균그룹`은 드롭다운에서 선택합니다.

```text
A
B
C
```

Grammar 평균은 일반 반 이름이 아니라 이 Grammar 평균그룹 기준으로 계산됩니다.

## 9. 출력설정 수정

`출력설정` 탭에서 시험/시즌마다 바꿀 수 있습니다.

```text
성적표 제목
코멘트 섹션 제목
출력월
저장폴더명
파일명규칙
```

예시:

```text
English Achievement Report – Final Semester 2026
English Achievement Report – Midterm Semester 2027
```

## 10. PDF 생성

현재 선택한 학생만 생성:

```text
성적표 생성 > 현재 행 학생 PDF 생성
```

출력여부가 TRUE인 전체 학생 생성:

```text
성적표 생성 > 전체 학생 PDF 생성
```

## 11. 시상 순위표 생성

성적 입력 후 아래 메뉴를 실행합니다.

```text
성적표 생성 > 시상 순위표 생성
```

그러면 `시상순위` 탭이 생성/갱신됩니다.

`시상순위` 탭에는 반별로 학생들이 총점 높은 순서대로 표시됩니다.

동점자는 같은 순위로 처리됩니다.

## 12. AI 코멘트 다듬기

짧게 입력한 코멘트를 학부모용 문장으로 다듬을 수 있습니다.

예시:

```text
스펠링 실수가 잦음
```

AI 다듬기 후:

```text
열심히 참여하고 있으나 스펠링에서 실수가 잦아 조금 더 꼼꼼한 확인이 필요합니다.
```

처음 한 번만 아래 메뉴에서 OpenAI API 키를 저장합니다.

```text
성적표 생성 > AI API 키 저장
```

그 다음 `성적입력` 시트에서 다듬을 학생 행을 선택하고 실행합니다.

```text
성적표 생성 > 선택 행 AI 코멘트 다듬기
```

여러 학생 행을 드래그해서 선택하면 여러 명을 한 번에 다듬을 수 있습니다.

전체 학생을 한 번에 검사하고 싶으면 아래 메뉴를 실행합니다.

```text
성적표 생성 > 전체 AI 코멘트 다듬기
```

전체 실행은 코멘트가 입력된 칸만 다듬고, 빈 코멘트 행은 건너뜁니다.

AI가 바꾸기 전 원문은 각 셀의 노트에 남습니다.

사용 모델은 `출력설정` 탭의 `AI 모델` 값에서 바꿀 수 있습니다.

## 13. 생성로그 확인

PDF 생성 결과는 `생성로그` 탭에서 확인할 수 있습니다.

성공한 경우 Google Drive PDF 링크가 기록됩니다.
